## （一）学习RNN相关知识

* 1.**RNN**
  * 适合处理**序列数据**（文本、语音、时间序列）
  * 三种结构：**多对多、多对一、一对多**
  * 优点：**参数共享**，大幅减少参数量，训练更高效
  * 缺点：**长期依赖问题**，梯度消失严重，无法记住很早之前的信息

* 2.**GRU（门控循环单元）**
  * 是 RNN 的改进版，用来**解决长期依赖缺陷**
  * 两个核心门：**更新门、重置门**
  * 更新门：控制保留多少**历史信息**
  * 重置门：控制忽略多少**过去无用信息**
  * 结构比 LSTM 简单，训练更快，效果接近 LSTM

* 3.**LSTM（长短期记忆网络）**
  * 解决 RNN **长期依赖与梯度消失**
  * 包含三个门 + 一个细胞状态：**遗忘门、输入门、输出门**
  * 遗忘门：决定丢弃哪些信息
  * 输入门：决定存入哪些新信息
  * 输出门：决定当前时刻输出什么
  * 细胞状态：像**“传送带”**，让信息远距离传递，不丢失

## （二）梳理了不同激活函数

* 1.sigmoid
  * 输出范围：0 ~ 1
  * 优点：平滑可导，适合二分类输出层，表示概率；缺点：容易梯度消失，输入值很大或很小时，导数接近 0
  * 适用场景：二分类任务输出层，隐藏层一般不使用

* 2.tanh
  * 输出范围：-1 ~ 1
  * 优点：**梯度更新方向更加自由**，收敛速度比sigmoid快；缺点：深层网络仍会出现梯度消失
  * 适用场景：RNN 隐藏层

* 3.ReLU
  * 公式：max(0, x)； x > 0 → 输出 x； x ≤ 0 → 输出 0
  * 优点：计算快（只需判断正负），正数区域导数恒为 1，基本不梯度消失；缺点：存在Dead ReLU，某些神经元永远输出 0
  * 适用场景：CNN、深度网络默认激活函数，隐藏层首选

* 4.LeakyReLU
  * 公式：max(αx, x)，α通常取0.01
  * 优点：保留ReLU优点，负区间有小梯度（负区间输出小于 0但 0），逼近解决神经元死亡问题；缺点：选择超参数α难调
  * 适用场景：防止ReLU死亡

* 5.ELU
  * 优点：负区间平滑，输出均值接近0，收敛更快，对噪声更鲁棒
  * 缺点：包含指数运算，训练速度慢
  * 适用场景：追求更高精度的网络结构

* 6.Swish
  * 公式：x·sigmoid(βx)
  * 优点：平滑非单调，在ImageNet等任务上优于ReLU；缺点：计算复杂
  * 适用场景：深层网络、Transformer
