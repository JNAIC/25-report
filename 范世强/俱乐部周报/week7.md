## 学习内容
 
### 7.1 深度卷积神经网络（AlexNet）
 
1. 核心定位
2012年ImageNet竞赛冠军模型，首次证明深度学习在计算机视觉领域超越传统手工特征，开启深度学习浪潮 。
2. 网络架构
整体共8层（不含输入层）：5个卷积层 + 2个全连接隐藏层 + 1个全连接输出层。- 卷积层：采用11×11、5×5、3×3不同尺寸卷积核，适配ImageNet高分辨率图像；卷积通道数为LeNet的10倍。
- 池化层：3×3最大汇聚层，步幅2，压缩特征尺寸。
- 全连接层：两层4096维隐藏层，参数量大，易过拟合 。
3. 关键创新- 激活函数：用ReLU替代Sigmoid，计算更简单、缓解深层网络梯度消失，加速收敛。
- 正则化：全连接层使用Dropout防过拟合；引入LRN局部响应归一化增强模型泛化性。
- 训练优化：GPU加速训练；数据增强（裁剪、翻转、变色）扩充数据集。
4. 与LeNet差异
更深、通道数更多；支持彩色高分辨率图像；新增ReLU、Dropout、LRN等机制；依赖GPU算力。
 
### 7.2 使用块的网络（VGG）
 
1. 核心思想
提出以重复块构建深度网络的设计范式，用固定3×3小卷积核堆叠加深网络，替代AlexNet的大卷积核，简化网络设计规则 。
2. VGG块（基础单元）
固定结构：多个3×3卷积（padding=1，保持尺寸）+ ReLU + 1个2×2最大池化（stride=2，尺寸减半）。
优势：相同感受野下，多层3×3卷积比单层大卷积核参数量更少、表达能力更强。
3. 典型架构（VGG-11）
5个卷积块 + 3个全连接层，共11层权重层 。- 卷积块配置： ((1, 64), (1, 128), (2, 256), (2, 512), (2, 512)) 。
- 通道数规律：每个卷积块后通道数翻倍，直至512后保持不变 。
4. 核心价值
奠定**“小卷积核+深层网络”**的设计基调，为后续ResNet、GoogLeNet等模型提供基础思路。
 
###  关键知识点对比总结
 
维度 AlexNet VGG 
卷积核 11×11、5×5、3×3混合 统一3×3小卷积核 
设计模式 手工调整层结构 重复VGG块堆叠 
激活函数 ReLU（核心创新） 沿用ReLU 
正则化 Dropout、LRN 沿用Dropout 
网络深度 8层 可灵活扩展（VGG-11/16/19） 
 
三、代码实现要点
 
1. AlexNet- 卷积层逐步缩小尺寸、扩增通道；池化层压缩特征图。
- 全连接层后接Dropout(0.5)降低过拟合。
2. VGG- 封装 vgg_block 函数，批量构建卷积块。
- 通过 conv_arch 超参数灵活控制块数与通道数，快速搭建不同深度VGG模型。
