# 第一份周报

## 一、本周学习内容

### Markdown 基本用法
本周学习了 Markdown 的基础语法
1. **标题语法**  
   使用 `#` 表示标题层级：  
   - `# 一级标题`  
   - `## 二级标题`  
   - `### 三级标题`
2. **文本格式**  
   - 加粗：`**文字**`  
   - 斜体：`*文字*`  
   - 删除线：`~~文字~~`
3. **列表**  
   - 无序列表：使用 `-` 或 `*`  
   - 有序列表：使用 `1. 2. 3.` 自动编号
4. **链接与图片**  
   - 插入链接：`[链接文字](网址)`  
   - 插入图片：`![图片说明](图片地址)`
5. **代码**  
   - 行内代码：使用反引号 `` `代码` ``  
   - 代码块：使用三个反引号包裹
6. **引用与分隔线**  
   - 引用：`> 内容`  
   - 分隔线：`---`
通过练习，已经能够独立编写结构清晰的 Markdown 文档。
