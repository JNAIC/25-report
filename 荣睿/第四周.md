# 学习报告
## 一、LeNet 网络结构概览

LeNet 是最早的卷积神经网络之一，常用于手写数字识别，典型结构如下：

1. 输入层：28×28 灰度图像
2. 卷积层 C1：6 个 5×5 卷积核 → 输出 6×24×24 特征图
3. 池化层 S2：2×2 平均池化，步长 2 → 输出 6×12×12 特征图
4. 卷积层 C3：16 个 5×5 卷积核 → 输出 16×8×8 特征图
5. 池化层 S4：2×2 平均池化，步长 2 → 输出 16×4×4 特征图
6. 卷积层 C5（全连接卷积层）：120 个 4×4 卷积核 → 输出 120×1×1 特征图
7. 全连接层 F6：84 个神经元
8. 输出层：10 个神经元，对应分类结果

---

## 二、特征图尺寸与通道变化

| 层 | 操作 | 输入尺寸 | 输出通道 | 输出尺寸 |
|----|------|----------|----------|----------|
| 输入 | - | 28×28×1 | 1 | 28×28×1 |
| C1 | 卷积 | 28×28×1 | 6 | 24×24×6 |
| S2 | 平均池化 | 24×24×6 | 6 | 12×12×6 |
| C3 | 卷积 | 12×12×6 | 16 | 8×8×16 |
| S4 | 平均池化 | 8×8×16 | 16 | 4×4×16 |
| C5 | 卷积 | 4×4×16 | 120 | 1×1×120 |
| F6 | 全连接 | 120 | 84 | 84 |
| 输出 | 全连接 | 84 | 10 | 10 |

## 三、关键模块解析

### 1. 卷积层（Convolution Layer）
- **功能**：提取局部特征，如边缘、角点、纹理
- **参数**：
  - 卷积核大小（kernel size）
  - 步长（stride）
  - 填充（padding）
  - 输出通道数（filters）
- **输出公式**：`(输入尺寸 - 卷积核尺寸 + 2*padding)/stride + 1`

### 2. 池化层（Pooling Layer）
- **功能**：降低特征图尺寸，减少计算量，保留主要特征
- **类型**：
  - 最大池化（Max Pooling）
  - 平均池化（Average Pooling）
- **参数**：池化核大小、步长

### 3. 全连接层（Fully Connected Layer）
- **功能**：整合特征，进行分类
- **特点**：每个神经元与上一层所有神经元连接

### 4. 激活函数（Activation Function）
- **功能**：引入非线性，提高网络拟合能力
- **常用函数**：
  - ReLU：0–∞，最常用
  - Sigmoid：0–1
  - Tanh：-1–1

## 四、PyTorch 代码示例
```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class LeNet(nn.Module):
    def __init__(self):
        super(LeNet, self).__init__()
        self.conv1 = nn.Conv2d(1, 6, kernel_size=5)   # C1
        self.pool = nn.AvgPool2d(2, stride=2)         # S2, S4
        self.conv2 = nn.Conv2d(6, 16, kernel_size=5)  # C3
        self.conv3 = nn.Conv2d(16, 120, kernel_size=4) # C5
        self.fc1 = nn.Linear(120, 84)                 # F6
        self.fc2 = nn.Linear(84, 10)                  # 输出层

    def forward(self, x):
        x = F.relu(self.conv1(x))  # 1×28×28 -> 6×24×24
        x = self.pool(x)           # 6×24×24 -> 6×12×12
        x = F.relu(self.conv2(x))  # 6×12×12 -> 16×8×8
        x = self.pool(x)           # 16×8×8 -> 16×4×4
        x = F.relu(self.conv3(x))  # 16×4×4 -> 120×1×1
        x = x.view(-1, 120)        # flatten
        x = F.relu(self.fc1(x))    # 120 -> 84
        x = self.fc2(x)            # 84 -> 10
        return x

# 测试网络
x = torch.randn(1, 1, 28, 28)
model = LeNet()
output = model(x)
print("输出 shape:", output.shape)  # torch.Size([1, 10])
```
