### 实战代码:computer:

跟着书里的代码敲了张量的基础运算，并重点测试了 `autograd` 自动求导功能，验证了框架算出来的梯度和数学公式推导的一致性：

```python
import torch

# ==========================================
# 1. Tensor 的基础操作与广播机制
# ==========================================
# 创建一个 3x1 的矩阵 A 和 1x2 的矩阵 B
A = torch.arange(3).reshape((3, 1))
B = torch.arange(2).reshape((1, 2))

# 测试广播机制：A 和 B 形状不同，但相加时会自动扩展为 3x2 的矩阵
C = A + B
print("矩阵 A:\n", A)
print("矩阵 B:\n", B)
print("广播机制相加后的结果 C:\n", C)


# ==========================================
# 2. 自动微分机制 (Autograd) 实战
# ==========================================
# 假设我们要求导的函数是 y = 2 * x^T * x
# 创建一个长度为 4 的向量，并设置 requires_grad=True 告诉系统需要存储梯度
x = torch.arange(4.0, requires_grad=True)

# 计算函数值 y
y = 2 * torch.dot(x, x)
print("\n计算得到的函数值 y:", y)

# 调用反向传播，自动计算 y 关于 x 的每个分量的梯度
y.backward()

# 数学推导：y = 2 * (x_0^2 + x_1^2 + x_2^2 + x_3^2)
# dy/dx = 4x
print("PyTorch 自动求导结果 x.grad:\n", x.grad)
print("数学公式理论结果 4*x:\n", 4 * x)

# 验证自动求导的结果是否和理论值完全一致
print("两者是否一致:\n", x.grad == 4 * x)
